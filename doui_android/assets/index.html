Tasks is a ToDo App

Inspired by David Allen's great book, "Getting Things Done", you can manage your tasks by Status, Category, or Context.

Tasks provides 5 lists to manage tasks based on WHEN they should get done:

  • Next: tasks that you should work on as soon as possible.
  • Waiting: tasks you are waiting for someone to work on.
  • Calendar: tasks that are scheduled.
  • Done: tasks you got done!
  • Someday: tasks you want to get to, but are not sure when.

Creating New Tasks

Create a new task by adding a new item to one of the Status lists (Next, Waiting, Calendar, Someday), add a Title, pick a Category, and Save.

Optionally, you can type a "@<person/place/thing>" as part of the Title or Description to automatically add this item to a Context list. Contexts are usually people, places, or even things, necessary to complete the task.

For instance, if we add a new item to the Next list, like this:

Title: Pick up flowers
Description: Remember @molly prefers red roses
Category: Family

The task will appear in 3 lists: Next, Family, and a list called "@molly".

New tasks can be added either to Status lists or to Categories.
